# Contributing

Thanks for helping build NSLAVE.

## How to contribute
1. Fork the repo
2. Create a branch: `feature/your-feature-name`
3. Make changes with clear commits
4. Open a Pull Request (PR) with:
   - What changed
   - Why it changed
   - Any screenshots (for UI)

## Guidelines
- Keep PRs small and focused.
- Use clear naming and avoid breaking changes without explanation.
- For docs updates: keep language simple and consistent with the project tone.

## Issues
Use labels:
- `docs`
- `web`
- `contracts`
- `security`
- `community`
