# Security Policy

## Reporting a Vulnerability
If you believe you found a security issue (website, API, or token-related documentation):

1. **Do not** open a public issue with exploit details.
2. Contact the maintainers privately:
   - Email: agent007@nslave.io

## Scope
- Website (`web/`)
- API (future)
- Token/addresses documentation (`contracts/`, `docs/`)

## Important Notes
- This project may include “coming soon” components. Treat unofficial links and addresses as untrusted.
- Always verify mint/treasury addresses from this repo once published.
